Listas de definicion

HTML
Html es un lenguaje de etiquetas de marcado para la creacion de paginas web sin ningun tipo de estilo.
CSS
Es un lenguaje de hojas de estilo en cascada que sirve para darle estilos a las paginas web, en pocas palabras sirve para ponerlas bonitas, XD.
JavaScript
JavaScript es un lenguaje de programacion orientado a objetos que sirve para darle dinamismo o interactividad a las paginas web, se utiliza tanto en el frontend como en el backend.
